Magento 2 Custom Order Number

Magento 2 Custom Order Number extension allows the store owner to change or customize any order number, invoice, or any other order-related details with any letter of prefixes, suffixes, or with any random number as per business requirements.

Also, Magento 2 change starting order number module attracts customers by showing them a more professional and attractive order number system with a unique code feature.

1. Easy for store management to update number as per their convenience.

2. Easy for making data report of their orders history.

3. More useful for visitors.

4. Compatible with any documents, easy change any series to set a unique one.

5. Show store high level of the management system that builds a reputation toward customers.
